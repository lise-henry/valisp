Valisp
======

() is equal to null
(if boolean expr-true) or (if boolean expr-true expr-false)

## Comparators ##

<, >, =, not=, <=, >= all work as you suppose and take 2+ arguments.

## Functions ##

(defn name type [params]
    expr)
Declare a function of type name, taking params. Params the form name type.

--------------
